# DC Motor Speed Control using PID Controller

## 📌 Description
This project simulates the speed control of a DC motor using a PID controller in MATLAB/Simulink. The controller adjusts motor input to achieve desired speed with minimal error.

## ⚙️ Features
- Closed-loop speed control
- PID controller implementation
- Adjustable Kp, Ki, Kd parameters
- Improved response (reduced overshoot and steady-state error)

## 🛠️ Tools Used
- MATLAB
- Simulink

## ▶️ How to Run
1. Open the .slx file in MATLAB
2. Run the simulation
3. Observe speed response in Scope
